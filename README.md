# Calculadora de pasos químicos - AdlCR

Calculadora para completar fácilmente los pasos químicos de Ataque de la Cosa Radiactiva.

Funcionamiento: 
1. Busca el número "PHI" en el mapa que no esté tachado y apuntalo. Introduce también la "M" que hay dentro de la estación de televisión.
2. Obten una batería y encuentra el cassete correcto. Apunta el compuesto correcto para derrotar a "la Cosa Radiactiva".
3. Cambia al color adecuado (obtenido en la televisión de Elvira) y busca los compuestos quimicos correspondientes
